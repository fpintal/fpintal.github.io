Section Steps Description
Code Editors 5 Learn how to set up and use code editors like VS Code. Covers installation, creating projects, using extensions, and productivity shortcuts.
Variables and Strings 62 Introduction to declaring variables, storing values, and manipulating text with strings. You’ll explore concatenation, template literals, and string methods.
Booleans and Numbers 37 Covers true/false values, comparisons, and number operations. Learn how JavaScript handles arithmetic, operators, and logical expressions.
Functions 38 Learn how to group reusable code into functions. Topics include function declaration, parameters, arguments, and return values.
Arrays 35 Explore how to store and manipulate collections of data. Includes array methods such as push, pop, shift, and iteration techniques.
Objects 28 Introduction to key–value data structures in JavaScript. Learn how to create, update, and access object properties.
Loops 22 Covers repetitive tasks using for, while, and do-while loops. Learn loop control techniques such as break and continue.
JavaScript Fundamentals Review 30 A recap of essential JavaScript basics including variables, operators, arrays, objects, and functions. Reinforces earlier concepts through practice.
Higher Order Functions and Callbacks 37 Learn about passing functions as arguments and returning them from other functions. Explore concepts like map, filter, and reduce.
DOM Manipulation and Events 87 Covers how to interact with web pages dynamically using the Document Object Model. Learn event listeners, selecting elements, and updating content.
JavaScript and Accessibility 42 Focuses on making interactive web content accessible. Learn techniques to ensure your JavaScript works well with screen readers and assistive tools.
Debugging 8 Learn strategies to find and fix errors in your code. Covers console logging, debugging tools, and common error types.
Basic Regex 47 Introduction to Regular Expressions. Learn how to match patterns in text using metacharacters, ranges, and quantifiers.
Form Validation 103 Build skills for validating user input with JavaScript. Covers handling required fields, patterns, custom messages, and interactive validation.
Dates 5 Learn to use the JavaScript Date object. Covers creating, formatting, and manipulating date and time values.
Audio and Video Events 55 Introduction to working with multimedia in the browser. Learn how to respond to audio and video events for custom interactions.
Maps and Sets 37 Learn modern ES6 data structures. Maps store key–value pairs with flexible keys, while Sets handle unique collections of values.
localStorage and CRUD Operations 78 Covers storing, retrieving, and updating data locally in the browser using localStorage. Practice CRUD (Create, Read, Update, Delete) operations.
Classes 68 Learn about object-oriented programming in JavaScript. Covers class syntax, constructors, inheritance, and methods.
Recursion 112 Introduction to recursion, where functions call themselves. Learn how to solve problems like factorials, traversals, and nested structures.
Functional Programming 23 Explore programming with pure functions, immutability, and higher-order functions. Covers map, reduce, and composition techniques.
Asynchronous JavaScript 36 Learn how JavaScript handles tasks that take time, like fetching data. Covers callbacks, promises, and async/await.
